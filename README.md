# pyrebel
# On Jetson Nano
Usage:
```python3 pynvrebel.py <filename.png> ```<br>
For example, 
```python3 pynvrebel.py letters.png```<br>

The expected input files are grayscale images of figures like <a href="https://github.com/ps-nithin/pyrebel/blob/main/letters.png">letters.png</a> or <a href="https://github.com/ps-nithin/pyrebel/blob/main/aaa.png">aaa.png</a> or <a href="https://github.com/ps-nithin/pyrebel/blob/main/sss.png">sss.png</a><br>
Running the above program will open a window showing the abstract points in the input image file as white points.
